# 分配

> [!NOTE]
> fwnet 内还会出现来自其他已互联的网络的数据，那些分配不受 fwnet 的控制。

[[toc]]

## IP

fwnet 使用以下 IP 段：

- `10.50.0.0/16`
- `fddd:5050::/32`

## ASN

fwnet 中用于 eBGP 的 ASN 范围为 **423737**0000 ~ **423737**9999。

## 域名

fwnet 使用以下内部专用域名后缀：

- `.fwnet`
- `.5050net` <Badge text="⛔ 停止新增" type="danger" />
