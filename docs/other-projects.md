# 其他项目

<!-- markdownlint-disable MD033 -->

以下是一些与 fwnet 类似的实验网络，其中部分网络已与 fwnet 互联。

## dn42

[dn42](https://dn42.dev/) 是一个大型、去中心化的网络。其目的是模拟一个互联网。它使用了大量在目前互联网骨干上应用的技术（例如 BGP 和递归 DNS），可以很好地模拟一个真实的网络环境。
